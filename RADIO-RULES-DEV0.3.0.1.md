# Règles radio DEV0.3.0.1

- Organisme contrôlé : dialogue autorisé.
- Approche / SIV / service d'information : dialogue autorisé selon la logique disponible.
- ATIS ou AWS : diffusion automatique, aucune réponse au pilote.
- Répondeur enregistré : lecture automatique, aucune conversation.
- Auto-information, CTAF ou UNICOM : PHONIE ne répond jamais au pilote.
- Fréquence inconnue : aucune réponse inventée.
