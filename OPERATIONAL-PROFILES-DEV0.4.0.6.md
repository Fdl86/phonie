# PROFILS OPÉRATIONNELS - DEV0.4.0.6

Les profils opérationnels restent nécessaires pour la carte, les diagnostics, les restrictions locales et la préparation du moteur multi-aérodromes.

À LFBI, le profil conserve :

- A3 : point intermédiaire ;
- A2 : point d'attente de départ ;
- A : entrée de piste.

La radio standard ne prononce plus ces noms. Le profil sert surtout à empêcher qu'un point intermédiaire soit accepté comme point de départ et à expliquer le chemin interne calculé.

Sur un aérodrome sans profil, PHONIE utilise la topologie Facilities. Un vrai HoldShort associé à la piste peut être accepté, tandis qu'aucun nom local n'est inventé à la radio.
